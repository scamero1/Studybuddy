Documento de Transparencia sobre Uso de IA – StudyBuddy

Resumen
La base del proyecto fue creada por Sebastian. A partir de esa base, Manuela y Juan David realizaron cambios sencillos acordes al nivel principiante en Java y refinaron la presentación web. Este documento detalla qué parte se generó con IA, qué modificaciones humanas se hicieron y cómo se comprobó que funciona correctamente.

Cómo se comprobó que funciona
- Java
  - Compilar: javac java/StudyBuddy.java
  - Ejecutar: java -cp java StudyBuddy
  - Pruebas manuales: agregar, listar, ver, resumen, palabras clave, ejercicios; confirmación de guardado en notes.dat.
- Python
  - Verificación de sintaxis: python -m py_compile python/studybuddy.py
  - Ejecución y uso del menú para operaciones básicas.
- Web
  - Servidor local: cd web y python -m http.server 8000
  - Navegar a http://localhost:8000/
  - Flujo completo: registro/login, crear/editar/eliminar notas, favoritos, filtros, búsqueda, exportar/importar, resumen, claves, ejercicios y práctica.

Alcance y limitaciones
- Resumen y palabras clave usan TF (frecuencia de términos) sin librerías externas; adecuado para nivel principiante.
- Autenticación web sin backend: contraseñas se guardan como hash en localStorage (no apto para producción).
- Apps funcionan en local (Java: serialización; Python: JSON; Web: localStorage y Web Crypto).

Cómo reproducir
- Java
  - Compilar: javac java/StudyBuddy.java
  - Ejecutar: java -cp java StudyBuddy
- Python
  - Ejecutar: python python/studybuddy.py
- Web
  - cd web
  - python -m http.server 8000
  - Abrir: http://localhost:8000/
